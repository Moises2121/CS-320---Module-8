How can I ensure that my code, program, or software is functional and secure?
In order to ensure the program is functional and secure, I must add tests that can run using different variables. In this course I learned how to integrate JUnit test cases that effectively test multiple inputs such as Null inputs, long and character-sensitive cases. Additionally, I learned how important it is to ensure test coverage meets the minimum requirements. By working in the backgorund using automated tests, I can ensure the code is secure before starting to work on the front-end development.

How do I interpret user needs and incorporate them into a program?
By following the requirements, I can ensure the needs are fulfilled. It is possible to have too much code that was not required, and not only will this have consequences on a timely manner project, but it will require additional test and resources that startup companies might not be able to afford. Having a close connection with the customer and following up on the status of the software, will help fulfill those needs.

How do I approach designing software?
Designing software could take some time, specially for new developers such as myself. I can approach each project by creating a progress checklist based on the requirements. By following these guidelines, I can move forward from each step of the software development lifecycle with ease and less stress, rather than guessing where I need to start.
